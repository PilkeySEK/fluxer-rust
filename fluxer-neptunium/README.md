# fluxer-neptunium

![fluxer-neptunium logo](https://raw.githubusercontent.com/PilkeySEK/fluxer-neptunium/refs/heads/master/.assets/fluxer-neptunium-logo-with-text-1.png)

Fluxer-neptunium is a Rust library/framework for the Fluxer API.

Check the [examples](https://github.com/PilkeySEK/fluxer-neptunium/tree/master/examples) for usage examples.

**This is a work-in-progress. Much of the API is not covered yet. Expect breaking changes.** Once the SemVer is at 1.* (not 0.*), much more features will have been added.
